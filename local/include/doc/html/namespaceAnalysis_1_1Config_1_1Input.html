int Analysis::Config::Input::nFiles = -1
static

Number of files on input.

Can be used to restrict the number of files to be read on input. It is ineffective if nEvents > 0, in which case the specified maximum number of events is read no matter what the number of files is (up to all available files if needed). It is effective together with nEvents = -1. The default value (-1) means all files.
