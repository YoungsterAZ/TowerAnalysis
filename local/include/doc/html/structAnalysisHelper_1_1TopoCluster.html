Generator for topo-cluster on a given energy scale.

Returns
fastjet::PseudoJet object with Analysis::FastJet::CaloSignalInfo object representing a topo-cluster attached. The returned energy and momentum correspond to a massless pseudo-particle with physically valid kinematics, even for unphysical detector signals.

The returned object has valid four-momentum kinematics. In case of a signal \( E < 0 \), the object is "ghosted", e.g. it has a tiny positive signal \( E \approx 0 \) but the correct \( (y,\phi) \) directions.

Parameters
datapointer to underlying data structure of templated type
idxindex of associated data
checkcheck on \( E > 0 \) if true, else generate fastjet::PseudoJet from all signals
sstrstring indicating energy scale (options "em" and "lcw", default "lcw")
Warning
In case an unknown string is passed to the function, an empty fastjet::PseudoJet object is returned.
